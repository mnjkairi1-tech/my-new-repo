# **App Name**: AI Atlas

## Core Features:

- AI Tool Search: Allow users to search for AI tools using keywords.
- Featured AI Tool: Display a featured AI tool prominently with an image background.
- Popular AI Tools: Show a list of popular AI tools with icons.
- Quick Access AI Tools: Provide quick access to AI tools through a grid-based layout with image cards.
- Mobile Bottom Navigation: Implement a fixed bottom navigation bar for mobile devices with Home, Tools, Community, and Profile options.

## Style Guidelines:

- Primary color: Light gray (#F5F5F5) for the background to provide a clean and modern look.
- Accent color: Soft black (#212121) for text to ensure readability against the light background.
- Highlight color: Yellow (#FFDA63) to emphasize interactive elements like the featured card border.
- Font: 'Inter' (sans-serif) for a modern and clean design, used for both headings and body text.
- Use rounded corners and soft shadows on cards to create a modern, soft UI.
- Implement a mobile-first responsive design optimized for desktop full-screen view.
- Employ icons from Lucide or HeroIcons for navigation and tool representation to maintain consistency and clarity.
